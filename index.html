Meteora

O projeto Meteora é um e-commerce de moda desenvolvido com foco em tecnologias como React, utilizando o Vite para build rápida e moderna, e Bootstrap para estilização. A aplicação aproveita a Context API junto com useReducer para gerenciar estados globalmente, garantindo um controle eficiente do carrinho de compras e otimizando o desempenho com useMemo. Esse projeto, em fase inicial, permite adicionar, remover e visualizar itens no carrinho, proporcionando um ambiente prático para explorar técnicas avançadas de gerenciamento de estado no ecossistema React.
